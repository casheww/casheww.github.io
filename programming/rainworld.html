Rain World

I do a lot of Rain World modding. My favourites of the mods I've worked on:

  • RegionCast, which adds Discord Rich Presence to the game. Webpage.
  • ToolBox - a small set of custom tools to help modders.
  • Slugs - some custom slugcat characters for the game.
  • SlugBrain - a WIP slugcat AI. Hopefully it will be convincing one day!

As well as writing mods for the game, I also help maintain some other Rain World related projects:

  • The Rain World Modding Wiki - repository.
  • The Rain World Wiki - CSS for our custom theme.
  • 5pebbles - a Discord bot that interfaces with the Rain World Wiki.
